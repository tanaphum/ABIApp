## Install Package

```{r}
install.packages("devtools")
devtools::install_github("slphyx/ABI")
```

or

```{r}
install.packages("devtools")
library(devtools)
install_github("slphyx/ABI")
```


## Use Package

```{r}
ABI::run_app_ABI()
```

or

```{r}
library(ABI)
run_app_ABI()
```
